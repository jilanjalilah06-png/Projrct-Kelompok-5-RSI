# 🌾 AgriConnect

AgriConnect adalah platform digital berbasis mobile dan web yang dirancang untuk membantu petani dalam memasarkan hasil pertanian, memperoleh informasi pertanian, serta mempermudah interaksi antara petani dan konsumen.

---

# 📌 Latar Belakang

Di era digital saat ini, sektor pertanian masih menghadapi berbagai kendala, seperti:

- Sulitnya pemasaran hasil panen
- Kurangnya akses informasi pertanian
- Keterbatasan komunikasi antara petani dan konsumen
- Pengelolaan data pertanian yang belum terintegrasi

AgriConnect hadir sebagai solusi digital untuk membantu proses pertanian menjadi lebih efektif, modern, dan terhubung.

---

# 🎯 Tujuan Project

- Membantu petani memasarkan hasil tani secara digital
- Mempermudah konsumen dalam mencari produk pertanian
- Menyediakan informasi dan konsultasi pertanian
- Mendukung digitalisasi sektor pertanian
